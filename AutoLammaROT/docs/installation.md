# Installation

## Stable release

To install AutoLammaRot, run this command in your terminal:

```sh
uv add AutoLammaRot
```

Or if you prefer to use `pip`:

```sh
pip install AutoLammaRot
```

## From source

The source files for AutoLammaRot can be downloaded from the [Github repo](https://github.com/TiagoUNB/Lamma).

You can either clone the public repository:

```sh
git clone git://github.com/TiagoUNB/Lamma
```

Or download the [tarball](https://github.com/TiagoUNB/Lamma/tarball/master):

```sh
curl -OJL https://github.com/TiagoUNB/Lamma/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd Lamma
uv pip install .
```
